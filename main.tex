\begin{SectionTable}{\headingfont Project and research experience}
November 2020 -- March 2021 &
\textbf{Progetto di Applicazioni e Giochi Multiplayer su Architettura Unity DOTS} \newline
Mentors: Prof. Paolo Bellavista, Dr. Andrea Garbugli (Alma Mater Studiorum). \newline
The thesis purpose was to analyze and outline the advantages of the new Unity Data-Oriented Technology Stack, compared to the traditional architecture of the game engine, which was based on a component model. Moreover, it was required the development of a complete multiplayer videogame prototype, with simple basic gameplay mechanics, built using DOTS.

GitHub repository \href{https://github.com/mikyll/UnityDOTS-Thesis}{here}. Presentation and summary of findings available \href{https://github.com/mikyll/UnityDOTS-Thesis/blob/main/Presentation/PresentationDOTS (pdf).pdf}{here}.\\

March 2020 -- June 2020 &
\textbf{Cluedo Game - Software Engineering Project} \newline
To pass the Software Engineering course we had to create e team of 3 and choose a topic to make the project document and a prototype. We decided to make an application for the murder mystery board game "Cluedo". Together we made the document through all the project phases, following the waterfall model, and we also submitted a presentation to show the work and choices done during the process. The game was coded entirely in Java (we used javafx for the graphics).

GitHub repository \href{https://github.com/mikyll/Cluedo}{here}. Presentation \href{https://github.com/mikyll/Cluedo/blob/master/Presentazione/Presentazione Cluedo (pdf).pdf}{here}.\\
